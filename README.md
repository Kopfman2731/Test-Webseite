🌐 Test-Webseite

A small test website project created to experiment with web development, Git workflows, and deployment on a web server.

This repository contains the source code for a simple static website that can be run locally or hosted online.

🚀 Features

Simple static website

Clean HTML structure

Easy to deploy

Lightweight and fast

Good for learning Git, HTML, and web hosting

📂 Project Structure
Test-Webseite/
│
├── index.html        # Main entry point of the website
├── css/              # Stylesheets
├── js/               # JavaScript files
├── assets/           # Images or media files
└── README.md         # Project documentation
🛠 Technologies Used

HTML5

CSS3

JavaScript

Git & GitHub

⚙️ Installation

Clone the repository:

git clone https://github.com/Kopfman2731/Test-Webseite.git

Navigate into the project folder:

cd Test-Webseite

Open the website:

index.html

or run a local server:

python3 -m http.server

Then open:

http://localhost:8000
🌍 Deployment

You can deploy this project on:

GitHub Pages

Apache

Nginx

Any static hosting provider

Example for Linux web server:

sudo cp -r * /var/www/html/
